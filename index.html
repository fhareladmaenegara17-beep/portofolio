Tentang Saya

Saya adalah pelajar SMK yang memiliki passion kuat di dunia jaringan komputer. Selama masa sekolah, saya aktif mengerjakan proyek-proyek riil, mulai dari konfigurasi router dan switch Cisco, Administrasi Server berbasis Linux (Debian/Ubuntu), hingga implementasi layanan jaringan seperti VoIP dan Web Server.

Tujuan saya adalah menjadi profesional di bidang keamanan jaringan atau network engineer.
